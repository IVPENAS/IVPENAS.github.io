  Throughout this 6-week course entitled Managing Enterprise Servers allows students to learn and gain proficiency in real-world industry scenarios to be prepared for managing enterprise servers using Linux Operating System along with practicing the DevOps culture. It allows users to configure services specifically using [1] Ansible which is an open-source tool that enables to construct Infrastructure as A Code (IaaC), monitor, and leverage alerting in the Development Operation environment while simultaneously collaborating with Git and GitHub, which is a control system that tracks new change, merging, synching, clones, and compares the codes created by the user.

[2] The concept of Containerization Technologies, which is a form of Operating System Virtualization or can be called a ‘Portable Computing Environment’ that allows Developers to deploy and create applications securely as the contained application is isolated user space-sharing in an OS and to transfer its data from one environment to another without having any errors. The students used Docker to containerize applications as its features specifically described its [2.1] Portability, which allows the developer to run the contained application in any platform or the cloud, [2.2] Agility, since containers have minimal capacities, it can incorporate its creation rapidly. [2.3] Scalability, and containerization can handle the increasing workload by reconfiguring the existing architecture to enable the resources with minimum resource usage. And [3] Cloud Services are commonly used by companies to provide services by aiding client’s incoming data with flexibility in order to keep up to the latest demand and supply from high to low without wasting and failing the resources of the cloud, and the money of both client’s and supplier. OpenStack is one of the Cloud Services that is mostly used by Administrators, It’s and students to hone their skills by controlling their respective data inputted to the cloud due to its open-source cloud computing operating system along with its Components namely: Horizon, Nova, Neutron, Cinder, Keystone, Glance and more services that allows Administrator to handle, store, complete projects efficiently.

Since then, student was able to complete each task that allows them to sharpen their critical thinking and problem-solving skills upon venturing in professionalism on managing enterprise servers by evaluating and solving real-world scenarios with the help of Educational Materials, Deployment Tools, and Management Applications in the most efficient way possible. This course serves as the stepping stone for the students to be a certified professional System Administrator, studying this course strengthens students' knowledge and confidence when it comes to evaluating and managing enterprise servers. Through the course, students can acquire and apply new knowledge as needed and develop strategies for future learning.
